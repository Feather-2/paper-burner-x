📖 使用说明

本地测试：
1. 启动 Worker：cd workers/academic-search-proxy && npx wrangler dev
2. 确认代理地址：http://localhost:8787

生产环境：
1. 部署 Worker 后，填入你的 Worker URL
2. 选择使用模式（推荐方案一）
3. 方案一：填入你自己的 API Key（可选）
4. 方案二：填入 Worker 的 Auth Key（必须）

测试范围：
- CrossRef、OpenAlex：直接访问（支持 CORS）
- PubMed、Semantic Scholar、arXiv：通过代理访问（解决 CORS）
